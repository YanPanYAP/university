# Лабораторная 7 (по практикуму ЛР №5): DES-EEE2 на Node.js

Реализовано приложение по варианту 4 (алгоритм `DES-EEE2`) с выполнением требований задания 5.2:

- разделение входного потока на 64-битные блоки;
- дополнение последнего блока (`PKCS#7`);
- преобразование ключевой информации (генерация раундовых подключей DES);
- шифрование/расшифрование `DES-EEE2` (`E_k1(E_k2(E_k1(M)))`);
- оценка скорости шифрования/расшифрования;
- пошаговый анализ лавинного эффекта (по битам исходного слова);
- анализ слабых и полуслабых ключей DES;
- оценка степени сжатия открытого и зашифрованного текста (`gzip`).

## Запуск

```bash
cd "/Users/yanpaulovich/Documents/fit/ИБ/lab 7"
npm start
```

## Что спрашивает программа при старте

1. Параметр преподавателя:
- `1` — обычные ключи (ввод `K1` и `K2` вручную);
- `2` — слабые ключи DES (выбор ключа из 4);
- `3` — полуслабые пары ключей DES (выбор пары из 6).

2. Источник текста:
- ввод вручную;
- загрузка из файла.

3. Количество итераций для измерения скорости.

## Выходные файлы

После запуска формируются:

- `output/report_lab7.txt` — текстовый отчет для сдачи;
- `output/results.json` — полный набор результатов в JSON;
- `output/cipher.bin` — шифртекст (бинарный);
- `output/decrypted.txt` — результат расшифрования.

## Примечание по корректности

В коде есть встроенный self-test:
- проверка DES на официальном тестовом векторе (`0123456789ABCDEF` / `133457799BBCDFF1` -> `85E813540F0AB405`);
- проверка обратимости `DES-EEE2` (encrypt/decrypt roundtrip).
